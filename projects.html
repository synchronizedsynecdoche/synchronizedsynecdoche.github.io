CryptoSnake and other stuff (write up more later)

My first project was Cryptosnake, a very small Python script that encrypted a user's string using AES in CBC mode with PBKDF2. I started creating a larger, more ambitious version of it that would fix some issues with the original version (such as not operating on bytes), but I haven't been working on it since school started. I also helped create a website for locating nearby charities at HackBCAIII alongside a group of 5 other people in 24 hours. I worked alongside Chris and Aditya at HackMIT to create ADBlockAR, an application that leverages Microsoft's HoloLens to remove advertisements in real life.
